# Tron Wallet for android

A Tron wallet for android.

## Refrence
- https://github.com/tronprotocol/wallet-cli

## Required.
 - android 4.1
 
## grpc and apk build
```
./gradlew build
```

## Feature
- Create Wallet
- Send Coin
- Receive Coin
- QR Code
