## Required.
 - android 4.1
 
## grpc and apk build
```
./gradlew build
```

## Features

- Create wallet
- Import wallet
- Send tron & token
- Receive tron & token
- My address QR code
- My tron balance
- My tokens
- My transfer history
- Vote representative
- Participate token
- Block explorer
- Token list
- Node list
- Representative list
- Tron market listing & price
