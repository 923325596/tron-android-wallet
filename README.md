## Required.
 - android 4.1
 
## grpc and apk build
```
./gradlew build
```

## Features

- Create wallet
- Import wallet
- Send tron
- Receive tron
- My address QR code
- My tron balance
- My tokens
- My transaction history
- Vote representative
- Token list
- Node list
- Representative list
- Tron market listing & price
